# Room booking REST API server
Nestjs server for room bookings demo project

Server is currently being hosted at http://ec2-3-17-36-231.us-east-2.compute.amazonaws.com

## Install Dependencies

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```
